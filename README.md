# RateSwitch X

Privacy-first currency exchange dashboard with zero tracking.

## Features

- Real-time currency conversion
- Interactive historical graphs (7D/30D/90D/1Y)
- Multi-currency converter
- Favorites system
- Offline-first functionality
- PWA installable
- Zero dependencies
- No tracking, no ads, no accounts

## Tech Stack

- Vanilla JavaScript
- Pure CSS
- SVG charts
- Service Worker
- LocalStorage
- Frankfurter API

## Keyboard Shortcuts

- `S` - Swap currencies
- `F` - Toggle favorite
- `M` - Toggle multi-currency
- `1-4` - Switch timeframes

## Performance

- Bundle size: <50KB
- Load time: <300ms
- Offline capable
- 15min cache duration

## Privacy

- No analytics
- No cookies
- No tracking
- No external fonts
- No fingerprinting
- 100% client-side

## Development

```bash
# Serve locally
python3 -m http.server 8000

# Or use any static server
npx serve .
```

## Deployment

Deploy to any static host:
- Vercel
- Netlify
- GitHub Pages
- Render
